TRACE

Visual Setup Guide

This guide walks you through deploying TRACE for your neighborhood chapter. No coding required. Total time: about 10 minutes.

What You Need

A computer with a web browser. That's it. Everything runs in the cloud on free tiers.

Step 1: Create Your Database

1 Go to neon.tech and sign up

2 Click New Project, name it anything, pick the region closest to your area

3 Copy the Connection String - this is your database address. Keep it private.

Neon provides free PostgreSQL databases with encryption at rest and TLS in transit. The free tier covers chapters up to 50 reporters.

Step 2: Deploy TRACE

1 Go to the TRACE GitHub page and click the Deploy with Vercel button

2 Sign in with GitHub (create a free account if needed)

3 Paste your database connection string when asked for DATABASE_URL

4 Click Deploy. Build takes about 2 minutes.

After deployment, Vercel gives you a URL like trace-abc123.vercel.app. Bookmark it - this is your chapter's home.

Step 3: Create Database Tables

1 Go back to Neon and open your project

2 Click SQL Editor in the sidebar

3 Open setup.sql from GitHub, copy the entire file

4 Paste into the SQL Editor, click Run

You'll see "CREATE TABLE" confirmations. "Already exists" messages are fine - the script is safe to run multiple times.

Step 4: Create Your Operator Account

1 Go to your-app.vercel.app/operator/

2 You'll see a First-Time Setup screen

3 Enter your chapter name (e.g., "Westside Watch")

4 Choose a callsign - your operator code name

5 Set an access code (6+ characters)

TRACE uses callsigns instead of email addresses to protect operator identity. Even if the database were compromised, attackers would find code names, not real people.

Step 5: Invite Reporters

1 In the operator console, click Admin

2 Go to Reporters

3 Enter a callsign for the reporter (code name only)

4 Click Generate Invite Code

5 Send the code to your reporter via Signal, WhatsApp, or in person

Step 6: Install on Phones

iPhone (Safari only)

Open your TRACE URL in Safari. Tap Share → Add to Home Screen → Add.

Android (Chrome)

Open your TRACE URL in Chrome. Tap three-dot menu → Add to Home Screen → Add.

The app installs from the browser - no App Store or Google Play required. No review process, no trail in purchase history.

Production Hardening

Before giving anyone access, set these environment variables in your Vercel dashboard:

TRACE_DISABLE_DEV_LOGIN=true - Disables the development login bypass

TRACE_DISABLE_TEST_CODE=true - Disables test invite codes

CORS_ORIGINS=https://your-app.vercel.app - Restricts API access to your domain

Need Help?

File an issue on GitHub or use the bug report button in the operator console sidebar.
